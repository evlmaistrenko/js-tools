      React hook that returns the nearest scrollable parent of a given element along a specified axis, or a fallback element if none is found.

      Parameters

      • element: null | HTMLElement

        The element whose scroll parent to find.

      • axis: ScrollParentAxis = "vertical"

        The scroll axis to consider: 'vertical', 'horizontal', or 'both'.

      • fallback: ScrollParentFallback = ...

        A function that returns a fallback DOM Element (never window).

      Returns { element: null | Element; lockScroll: () => void; unlockScroll: () => void }

      • element: null | Element

        DOM element that is scrollable

      • lockScroll: () => void

        Function to lock scrolling on the scroll parent
