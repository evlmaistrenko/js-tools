# Icons.

Some icons witch Ant Design does not have yet.

## Usage.

```javascript
import { TelegramFilled } from "@evlmaistrenko/js-tools-antd/icons";
import { Button } from "antd";

// Anywhere you want:
<TelegramFilled />;

// Or i.e. in Button:
<Button icon={<TelegramFilled />} />;
```
